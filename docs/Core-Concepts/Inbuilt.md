# Inbuilt Methods

Ethereal comes with a few built in methods.

## 1. log([, ...])

log is a function that prints the given value to the console.

**Example:**
```etrl
log("Hello, world!");
```

Log can have multiple parameters.

**Example:**
```etrl
log("Hello, world!", "This is a message");
```
