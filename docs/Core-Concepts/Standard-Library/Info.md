# Info

Ethereal comes with a few built-in libraries.

They are:

1. `std:util`
2. `std:fs`
3. `std:array`
4. `std:string`
5. `std:math`

Standard Libraries can be imported using the `include` keyword followed by the name of the library.

**Example:**

```etrl
include "std:util";
```
